# Chatbot RAG Gemma-2B-IT

A Retrieval-Augmented Generation (RAG) chatbot built using Gemma-2B-IT and Google Colab.  
This project focuses on integrating document retrieval with Large Language Models (LLMs) to improve response accuracy and contextual understanding.

---

## Features

- Retrieval-Augmented Generation (RAG)
- AI-powered chatbot responses
- Document-based question answering
- Google Colab implementation
- Basic conversational workflow
- Large Language Model integration

---

## Technologies Used

- Python
- Gemma-2B-IT
- Google Colab
- LangChain
- Hugging Face
- FAISS
- Transformers
